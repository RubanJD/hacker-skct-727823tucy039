# Password Manager with Encryption

## Description
This project is a simple password manager that securely stores and retrieves passwords using encryption.

## Features
- Save password
- View password
- Encryption using Fernet

## Tools Used
- Python
- Cryptography library

## How to Run

1. Install requirement:
pip install cryptography

2. Generate key:
python key_gen.py

3. Run program:
python tool_main.py
